HTML 示例文档

使用说明

在这个给出一份简单的html示例，包涵大多数html标签，以及其使用方法。 网络上有很多关于html的描述，但是通常很多是把HTML4的一些特性混在了 一起。实际上，我们只需要学会简单的HTML5标准就行。

作为为HTML代码使用的一个参考，可以直接把这些代码拷贝到自己的项目 中就可以。作为学习使用，主要在这个文档中找到相应的标签，这里的使用 方法是现代所推荐的用法。也可以作为一个大家选择看书的一个标准，看 计算机相关的书肯定得看书中代码的。

对于这份示例文档，我们仅仅需要能够看懂每一部分就行了，其他的大家需要自己 扩展一些内容。比如最顶上的doctype声明，这还是有的一段历史的，我给大家看 的之是一个现在最合适的，如果你想明白为什么很多现在的网页中并不如此，你需 要自己使用Google或者Baidu。通常对于网络的学习，我们仅仅需要有搜索引擎和 一个能够分别是非的大脑就好了。

代码规范

代码就像是书写一篇文章，写得好的不仅格式优美，内容也是简洁规范， 而且，对人友好。当然，作为一门标记语言，HTML本身也比较简单，要求 不是那么严格。这里扩展一下优秀代码的几个标准，同时也作为一个 自己写程序的一个规范准则吧。

扩展

HTML5包含很多新的特性，比如HTML视频。这些新的特效在现代的浏览器 中还没有完全被实现。这些大家有兴趣可以去研究一下。

源码


<!DOCTYPE HTML>
<html lang="zh-cn">
<head>
	<meta charset="UTF-8">
  <link rel="stylesheet" href="style/foo.css" />
  <script src="js/foo.js" type="text/javascript"></script>
	<title>HTML示例文档</title>
	
	<style type="text/css">
   ul.foo{
      border:1px solid green;
   }
   /*这是在css文件中导入外部css文件的方法，import*/
   @import "../foo.css";
  </style>
  
  <script type="text/javascript">
    //这里引入js
    function foo(){
      alert('Hello,world. This is my first js function');
    }
  </script>
 
</head>
<body>
 

  <div id="page">
  
    <div id="header">
      <h1>标题1</h1>
      <p class="p1">
        段落，h1是标题1，从1到6，级别原来越不重要。通常，我们用h表示一个
        标题，对段落的总结。段落是普通的文本，在大片文字内容中是主体部分。
      </p>
      
      <ul class="navigation">
        <li>
          <a href="linkto/index.html" class="test1" title="内部链接,外部打开" target="_blank">首页</a>
        </li>
        <li>
          <a href="http://localhost/" class="test2" title="title属性是鼠标点击过来时显示的提示">外部链接</a>
        </li>
        <li>
          <a href="mailto:example@gmail.com" class="test3" title="">非链接</a>
        </li>
      </ul>
      
    </div>
    
    <div id="main">
    
      <div id="col-extra">
        <h3>导航</h3>
        <dl>
        	<dt>定义列表，有时用于导航</dt>
          <dd id="left-nav-1"><a href="#">items 1</a></dd>
        	<dd id="left-nav-2"><a href="#">items 2</a></dd>
        	<dd id="left-nav-3"><a href="#">items 3</a></dd>
        	<dd id="left-nav-4"><a href="#">items 4</a></dd>
        	
        	<dt>定义通常只有两个级别，ul可以循环嵌套</dt>
        	<dd class="left-nav-2-1"><a href="#">items 1-1</a></dd>
        	<dd class="left-nav-2-2"><a href="#">items 1-2</a></dd>
        	<dd class="left-nav-2-3"><a href="#">items 1-3</a></dd>
        </dl>
      </div>

      </pre>

          </thead>

            </tr>

            	<td>col1-2</td>
            	<td>col1-3</td>

            	<td>col3-3</td>
            </tr>
          <tbody>
          
          </tbody>
          
        </table>
        

        <ul>
          <li>列表元素</li>
          <li>
            这里使用的主要的是链接元素的属性，头部通常有一个链接，链接到主页。title
            属性有title和target，对于HTML5标准中，任何属性如果为空，一定要使用空双
            引号表示。属性与属性之间需要用空格分开。
          </li>
          <li>

              <li>_blank -- 在新窗口中打开链接</li>
              <li>_parent -- 在父窗体中打开链接</li>
              <li>_self -- 在当前窗体打开链接,此为默认值</li>

          </li>

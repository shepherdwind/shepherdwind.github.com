css与html基础

本章主要介绍html基础，

html
Hypertext Markup Language，简称为HTML）是为“网页创建和其它可在网页浏览器中看到的信息”设计的一种置标语言。HTML被用来结构化信息——例如标题、段落和列表等等，也可用来在一定程度上描述文档的外观和语义 [1] 。
css
Cascading Style Sheets（层叠样式表／串样式列表），是html的表现形式。

html

html是一种文档，所谓的文档是记录文字的文件，我们日常用得比较多的 有word文档和txt文档。这些文档是我们所见即所得的文档，通常形式和 内容混合在一起，word文档如果失去了其格式，和txt文本文档是一样的。 学习html，我们首先需要理解计算机文档。

虽然上面维基百科中，把html解释为一种[http://zh.wikipedia.org/zh-cn/置标语言 标记语言]，但是，现在的html解释为一种文档更合适。这意味着:

  1. 我们写文档也可以用html来写，比如：写论文，一般的笔记摘抄。这些都可以用html来写， word软件提供不少方便，但是，一旦离开了office软件，你的文档就基本不能 用。html是纯文本的，所以你永远不用担心这个。当然，更重要的是，HTML具有 着非常优秀的表现力。我现在所写的文档即是HTML模式。
  2. 其次，我们切网页的时候，需要怀着这样的想法：自己是在写一份关于网站介绍的 文档。我们需要写出优秀的HTML文档，而不管这个页面是什么样子的。这里需要做 到你的HTML结构清晰简洁，可阅读性强。

html基本结构

HTML由三部分构成，

  1. 标签,比如span div head title等等，每个标签都有其特点的含义 的含义。安文档流顺序，最上面是html类型声明 <!DOCTYPE HTML>， 这个是给浏览器看的，让其知道以何种方式来解释文档。然后是html 标签，html标签又包含head和body标签。下面是一份html的基本结构。 
      <!DOCTYPE HTML>
  <html lang="en">
  <head>
  	<meta charset="UTF-8">
  	<title></title>
  </head>
  <body>
  	
  </body>
  </html>
  2. 属性,html5所保留的属性很少，记住任何表现形式的属性都改由css实现了 html标签的属性主要有以下三类
    1. 通用的id和class属性，任何body下的标签都可以设置。
    2. 外部链接属性。其他还有img,link,a等链接元素的属性，指向外界, 这里的外界指 的是html文档本身以外的任何文件，图片，css文档或者js。 外部链接分为绝对路径和相对路径，前者为完整的url路径，或者本地的完整路径 比如：D:/my document/AppSer/www/index.html,绝对路径比较少用，因为一个 文档或者网站本身的位置是经常发生变化的。如果引入的外部链接越多，文档就 越依赖于外部环境的稳定。win下常用反斜杠\作为路径，更推荐使用斜杠/,在很多 语言中，反斜杠作为转义字符。 相对路径 比较常用，只需要记住，相对路径以本文件为起点，./，指代本文件夹， ../,指代上级路径。这个可以在rar压缩文件中看到，在linux看到就更简单了。在文 件系统中，文件名是.的就是指代本文件夹，..指代父目录。
    3. 表单元素的属性。表单元素比较常用，自身所拥有的属性也比较多，主要是用于形容表单 元素是什么。
  3. 内容,内容通常是我们看到的。这个没有什么可以说的，内容通常是文字，需要注意的一点 是，内容可以包含图片等多媒体，但是，很多时候，图片只是背景，是一种网页的展示 形式，不能放在HTML文档中。比较常见的一个问题就是html中图片替换问题。

杂项

其他的，html需要注意的是html声明，声明说明了html是什么。这个是一个历史的问题了。 html从4.0到现在的5.0，4.0以前的我都没有听说过。中间有一个4.1过度版本,不同版本之间 的区别不是很大。现在我们需要使用的就是html5版本的，这个声明很简单<!DOCTYPE HTML>
